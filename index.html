Visual Studio is a platform on which you can build online games.
If you want to create websites, then Brackets IDE will suffice.
But not for games. To set it up do the following steps in the following order:

1. Download Visual Code Studio on your PC.

2. Open VCS and download the "Live Test" application.

3. Create a folder and start creating!



*Now, to open the console, right click and choose "Inspect"*
